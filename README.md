## 概要
「ミートアンドイート」は、参加者の日程と位置情報を集約し、全員が集まりやすい最適なレストランを自動で決定するサービスです。

## 主な機能
- **日程調整**: 幹事が登録した候補日に対し、参加者が回答・集計。
- **位置情報の取得**: 幹事および参加者の現在地を取得。
- **最適地点での検索**: 取得した位置情報を元に、全員の中間地点付近でレストランを提案。

## 利用API
- **Google Routes API (Compute Route Matrix Pro)**
  - 全員の出発地からの中間地点を高度に計測。
- **ホットペッパー API (リクルート)**
  - 中間地点周辺の飲食店情報を取得。

## サービスの流れ
1. **設定**: 幹事が参加人数と候補日程を登録。
2. **回答**: 参加者が日程調整に回答し、位置情報を送信。
3. **決定**: 全員の位置情報を元に、中間地点にあるレストランを自動で選定。
